# Code Book

## ID Fields

* `subjectId` - The participant ("subject") ID
* `activity` - The label of the activity performed when the corresponding measurements were taken
** `WALKING`
** `WALKING_UPSTAIRS`
** `WALKING_DOWNSTAIRS`
** `SITTING`
** `STANDING`
** `LAYING`
